# Behaviour
- Partition(directory)
     - Chunk(voxel group, voxel = cube)
          - Voxel(save a data)
              - Data(information/data)

Example:
- Partition name: Ideas
- Chunk [3**3(27 voxels)] name: Website
- Voxels [0,1,2,3,4,5,6,7...27]:
- Voxel 0: Store website
- Voxel 1: Communication website
- [...]
